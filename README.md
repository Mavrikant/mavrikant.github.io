## Tech Stack

This site is built with [Jekyll](https://jekyllrb.com/) using the [Clean Blog](https://startbootstrap.com/themes/clean-blog-jekyll/) theme by Start Bootstrap.

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Mavrikant/mavrikant.github.io.git
   cd mavrikant.github.io
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Run the site locally:
   ```bash
   bundle exec jekyll serve
   ```

4. Open your browser at `http://localhost:4000`
